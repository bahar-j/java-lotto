### 기능 요구사항
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.  
- 로또 1장의 가격은 1000원이다.
- 로또 1장은 1~43 사이의 6개의 숫자로 구성된다.
- 로또 당첨 번호 6개를 입력받는다.
- 3개 일치(5천원), 4개 일치(5만원), 5개 일치(150만원), 6개 일치(20억)
- 당첨 통계와 수익률을 보여준다.
